# Lab Guide Directory

This folder contains the guided materials used to walk you through the hands-on exercises.

## What’s in This Folder?

The files in this directory are designed to:

* Provide step-by-step lab instructions
* Frame each hunt with a clear objective
* Introduce hypotheses to test
* Offer hints
* Guide you through analysis techniques

---

## How to Use the Lab Guide

* Follow the files introduced by the instructor.
* Read the scenario carefully before writing queries.
* Form a hypothesis before analyzing the data.
* Document your findings as you go.

The guide is meant to assist your thinking — not replace it.

---

## ⚠️ Important

This is a hands-on threat hunting lab.

You are expected to:

* Ask investigative questions
* Write and refine your own queries
* Validate findings before drawing conclusions
* Defend your assessment

The lab guide provides direction — the analysis is yours.

Happy hunting!
